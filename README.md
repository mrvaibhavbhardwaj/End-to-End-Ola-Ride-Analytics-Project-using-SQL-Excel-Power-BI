🚖 Ola Ride Booking Analytics Dashboard

Project Overview

This project analyzes ride booking data from an Ola dataset to uncover operational and customer insights. The data was explored and analyzed using SQL and visualized using Power BI to create an interactive dashboard.

The goal of this project is to identify booking trends, ride performance, cancellation patterns, and customer behavior through data visualization.

---

🎯 Objectives

* Analyze total ride bookings and ride trends
* Compare successful vs cancelled rides
* Identify demand for different vehicle types
* Understand ride distance distribution
* Create an interactive dashboard for better business insights

---

🛠 Tools & Technologies

* Microsoft Power BI
* SQL
* Microsoft Excel
* Data Visualization
* Data Cleaning & Transformation

---

📈 Dashboard Features

* Total Bookings KPI
* Successful vs Cancelled Rides Analysis
* Vehicle Type Demand Analysis
* Ride Distance Distribution
* Interactive Filters and Visualizations

---

🔍 Key Insights

* Most ride bookings were successfully completed, while a portion were cancelled by drivers or customers.
* Certain vehicle types showed higher demand compared to others.
* Short to medium ride distances dominated the majority of bookings.
* Booking patterns can help improve driver allocation and operational efficiency.
